# Guia de Ejercicios Fisicos

Bienvenido a nuestra guia de ejercicios fisicos, donde compartimos descripciones y beneficios de diferentes tipos de ejercicios.

## Tipos de Ejercicio

- [Ejercicio Cardio](ejercicios/cardio.md): Mejora tu resistencia y salud cardiovascular.
- [Ejercicio de Fuerza](ejercicios/fuerza.md): Aumenta la masa muscular y la fuerza.
- [Ejercicio de Flexibilidad](ejercicios/flexibilidad.md): Mejora la movilidad y reduce el riesgo de lesiones.

Esperamos que encuentres utiles estos ejercicios y te inspires para mantenerte activo y saludable
